Feature Engineering

In the Feature Engineering section, new features are created from the 'DateHour' column that could significantly impact the model's predictive capabilities.

Here is an analysis for each engineered feature:

  • Year Extraction: By introducing the 'Year' feature, the model can learn from these annual variations, potentially improving its accuracy in making predictions or recognizing trends tied to specific years.

  • Month Extraction: The extraction of the month introduces a cyclical aspect to the dataset, capturing seasonal trends and monthly variations. This enhances the model's ability to predict outcomes influenced by seasonal factors.

  • Day Extraction: The 'Day' feature, derived from the date, captures daily fluctuations within the data and is useful for datasets where changes occur throughout the month, enabling the model to discern patterns that occur on a day-to-day basis.

  • Hour Extraction: By extracting the hour from 'DateHour', this feature introduces the time-of-day aspect, which is critical for datasets where time significantly influences the observed outcomes. The feature allows the model to capture diurnal patterns, such as peak activity hours, potentially improving its predictive performance in scenarios where time of day is a determining factor.

  • DateHour Column Removal: After extracting these temporal components, the original 'DateHour' column is dropped to avoid redundancy and potential confusion for the model.

Extracting the date and hour from the ‘DateHour’ column and creating separate features for year, month, day, and hour. This can help capture the seasonal, weekly, and daily patterns of bike demand

## Feature Engineering ##

# Convert Datehour to datetime data type
df_full['DateHour'] = pd.to_datetime(df_full['DateHour'])

# Extract year
df_full['Year'] = df_full['DateHour'].dt.year

# Extract Month
df_full['Month'] = df_full['DateHour'].dt.month

# Extract Day
df_full['Day'] = df_full['DateHour'].dt.day

# Extract Hour
df_full['Hour'] = df_full['DateHour'].dt.hour

# Drop DateHour column
df_full.drop('DateHour', axis=1, inplace=True)
